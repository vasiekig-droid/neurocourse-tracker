☁️ GitHub Sync: телефон + компьютер

Это та самая схема с токеном: приложение сохраняет данные в отдельный файл репозитория, а телефон и компьютер читают один и тот же прогресс. Токен хранится только в браузере на твоём устройстве, не в коде сайта.
GitHub Sync ещё не настроен. Заполни поля, сохрани настройки, потом нажми «Сохранить в GitHub» на компьютере и «Загрузить из GitHub» на телефоне.
Важно: не вставляй токен прямо в файл index.html перед публикацией. Вводи его только в поле на открытом сайте. Для токена достаточно доступа Contents: Read and Write только к этому репозиторию.

🔁 Резервная копия вручную

Это запасной способ, если GitHub Sync временно капризничает. Можно скачать backup и импортировать его на другом устройстве вручную.
Не публикуй backup в открытый GitHub, если будешь писать туда личные заметки. Сам файл приложения можно публиковать спокойно.
