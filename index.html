🧩 Планирование, выходные и занятость

Здесь можно заложить выходной, добавить параллельную занятость и подобрать лучшие окна для учёбы. Эти данные тоже уходят в GitHub Sync.

🧯 Анти-выгорание

Настрой мягкий предел, после которого трекер начнёт показывать «хватит на сегодня». Это не запрет, а заботливый ограничитель, чтобы курс не превратился в наждачку для мозга.

🧱 Моя параллельная занятость

Добавляй разовые дела или повторяющиеся блоки: танцы по понедельникам и четвергам, английский, смены детей, сон, готовка, семейное время, поездки.

➕ Разовая занятость

🔁 Повторяющаяся занятость

Выбери период и дни недели. Трекер сам разложит занятость по календарю и будет учитывать её при подборе окон для учёбы.

✈️ Поездка / плотный период

Для поездок, каникул, болезни, плотных семейных дней. Можно заблокировать весь день и/или сделать дни выходными для учёбы.

📥 Импорт занятости из Google Calendar

Если импорт снова капризничает, можно не мучить эту дверь: повторяющиеся дела выше заменяют календарь внутри трекера.
