Как обновить GitHub Pages:
1. В репозитории замени файлы index.html, manifest.json, sw.js.
2. Нажми Commit changes.
3. Открой сайт по ссылке с антикэшем: https://vasiekig-droid.github.io/neurocourse-tracker/?v=36

Важно: токен GitHub не вшит в файл. Он хранится только в браузере устройства.
