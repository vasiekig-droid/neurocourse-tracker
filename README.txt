Как обновить GitHub:
1. Открой репозиторий neurocourse-tracker.
2. Замени index.html, manifest.json и sw.js файлами из этого архива.
3. Нажми Commit changes.
4. Открой сайт с параметром: https://vasiekig-droid.github.io/neurocourse-tracker/?v=39
