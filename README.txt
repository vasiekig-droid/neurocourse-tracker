НейроКурс Трекер v3.8

Что обновлено:
1. Убраны быстрые кнопки +10/+15/+20/+30 на главной карточке плана.
2. Добавлен выбор, в какой урок записывать просмотр.
3. Вместо прибавки минут теперь можно вписать итоговую позицию, где остановилась: например 15:04 или 1:05:00.
4. В блоке последних дней появились кнопки удаления конкретного дня просмотра.
5. Для новых добавлений просмотра появилась история последних добавлений с удалением.
6. Импорт календаря теперь подписан как .ics/.ical и принимает .ical-файлы.
7. В планировании теперь видно не только занятость выбранного дня, но и всю ближайшую добавленную занятость.
8. Обновлен PWA-кэш до neurocourse-tracker-v38.

Как загрузить на GitHub:
1. Открой репозиторий neurocourse-tracker.
2. Замени index.html, manifest.json и sw.js файлами из этого архива.
3. Нажми Commit changes.
4. Открой сайт по ссылке:
   https://vasiekig-droid.github.io/neurocourse-tracker/?v=38

Важно:
Токен GitHub не встроен в файл. Его нужно ввести на каждом устройстве в настройках синхронизации.
