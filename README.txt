НейроКурс Трекер 3.3

Что нового:
- Облачная синхронизация между телефоном и компьютером через GitHub.
- Прогресс сохраняется в progress.json в этом же репозитории.
- Весь курс встроен в index.html.

Как обновить GitHub:
1. Распакуйте архив.
2. Загрузите index.html, manifest.json, sw.js, README.txt в репозиторий поверх старых файлов.
3. Нажмите Commit changes.
4. Откройте сайт с ?v=33

Как включить синхронизацию:
1. В приложении нажмите «☁️ Синхронизация».
2. Введите:
   - GitHub логин
   - neurocourse-tracker
   - main
   - GitHub token с доступом Contents: Read and write к этому репозиторию
3. Нажмите «Сохранить настройки».
4. На устройстве с актуальным прогрессом нажмите «⬆️ В облако».
5. На втором устройстве нажмите «⬇️ Из облака».

Важно:
Токен хранится только в браузере устройства. Не публикуйте его в репозитории и не отправляйте другим людям.
