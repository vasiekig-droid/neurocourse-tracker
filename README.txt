Как обновить GitHub:
1. Распакуйте архив.
2. В репозитории neurocourse-tracker откройте Add file → Upload files.
3. Загрузите все файлы из папки: index.html, data.js, manifest.json, sw.js, icon.svg, README.txt.
4. Нажмите Commit changes.
5. Подождите 1–3 минуты и обновите сайт.
6. Если телефон показывает старую версию, закройте приложение, откройте снова или обновите страницу в браузере.

Важно:
Прогресс хранится в браузере конкретного устройства. Для переноса между телефоном и компьютером используйте «Бэкап» и «Восстановить».
