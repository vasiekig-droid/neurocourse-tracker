# НейроКурс Трекер

Это мини-приложение для учёта прохождения курса.

## Как открыть на компьютере
Откройте `index.html` двойным кликом.

## Как установить на телефон
Для полноценной установки как приложения PWA папку нужно разместить на HTTPS-хостинге:
- GitHub Pages
- Netlify
- Vercel
- любой другой сайт с HTTPS

После открытия ссылки на телефоне:
- Android Chrome: меню ⋮ → «Добавить на главный экран» / «Установить приложение»
- iPhone Safari: «Поделиться» → «На экран Домой»

## Важно
Прогресс хранится в браузере устройства. Для переноса используйте кнопки «Экспорт» и «Импорт».
