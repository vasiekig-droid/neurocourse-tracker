Как обновить GitHub Pages:
1. В репозитории замени index.html, manifest.json и sw.js файлами из архива.
2. Нажми Commit changes.
3. Открой сайт с параметром: ?v=37

Как импортировать Google Calendar:
1. В Google Calendar экспортируй календарь в .ics.
2. Открой вкладку “Планирование”.
3. Нажми “Загрузить .ics”.
4. События появятся как занятость и будут учитываться при подборе окон для учёбы.
