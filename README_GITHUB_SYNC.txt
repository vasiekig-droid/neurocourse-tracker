НейроКурс Трекер v3.4 — GitHub Sync

Что делать:
1. Открой репозиторий на GitHub.
2. Замени старый index.html на новый index.html из этого архива.
3. Нажми Commit changes.
4. Открой сайт GitHub Pages на компьютере.
5. В блоке GitHub Sync введи owner, repo, branch, tracker-data.json и токен.
6. Нажми «Сохранить настройки», затем «Сохранить в GitHub».
7. Открой сайт на телефоне, введи те же настройки и нажми «Загрузить из GitHub».

Важно:
- Токен не вставляй в код index.html. Вводи только в поле на сайте.
- Лучше использовать fine-grained GitHub token только для одного репозитория с правом Contents: Read and Write.
- Файл данных по умолчанию: tracker-data.json. Его приложение создаст само.
